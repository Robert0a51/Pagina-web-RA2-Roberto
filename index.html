Introducción

Este trabajo servirá para demostrar los conocimientos adquiridos en HTML y CSS, incluyendo el uso de elementos semánticos, listas, tablas, formularios, imágenes, enlaces, estilos, modelo de caja y Flexbox.

La estructura de esta página se basa en:

      Un apartado de Introduccion
    • Nombre y título o frase que te identifique.
    • Imagen o logo personal.
    • Menú de navegación con enlaces internos a las secciones.
      Un apartado de descripción personal
    • Breve texto descriptivo.
    • Puedes incluir una imagen o datos personales (pueden ser ficticios).
      Un apartado de descripción de aficiones
    • Muestra tus intereses o trabajos.
      Un apartado de descripción de Habilidades
    • Lista de habilidades o competencias.
      Un apartado de descripción de Contacto.
    • Formulario con al menos los siguientes campos: nombre, correo electrónico y mensaje. Se valorará si se usa otros tipos de controles de formulario.
    • Enlaces a tus redes sociales (pueden ser enlaces vacíos)

Se deberá de usar Flex en al menos 3 ocasiones. Ya sea para centrar un elemento, para distribuir el menú o para generar una galería con tarjetas de vuestros conocimientos.
